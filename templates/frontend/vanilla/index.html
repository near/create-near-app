Look at that! A Hello World app! This greeting is stored on the NEAR blockchain. Check it out:

  1. Look in index.js - you will see a Wallet object being created and used to interact with the methods set_greeting and get_greeting.
  2. These method live in the smart contract, who's code is defined in the contract/ folder.
  3. When you run npm run start, the code in contract/ gets deployed to the NEAR testnet. You can see how this happens by looking in package.json at the scripts section to find the dev command.
